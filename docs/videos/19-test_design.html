Concepts:

  • What test to write first? Write the test that forces you to write the code that you already know you want to write.
  • Given, When, Then
  • One way to name tests is to use the Action (the When) and the Assertion (the Then). Example: whenPayrollIsRun_thenPaymasterWillHoldCheckForHoursWorkedTimesHourlyRate
